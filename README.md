# Deutsch Für Alle

A modern, responsive website for a German language institute, built with Next.js, React, TypeScript, and Tailwind CSS.

## 🌐 Live Demo

https://deutsch-fur-alle.vercel.app/

## 📂 GitHub Repository

https://github.com/Hafsasohail25/deutsch-fur-alle

## ✨ Features

- Modern responsive design
- Course information
- Hero section
- Contact section
- Mobile-friendly layout
- Fast performance

## 🛠 Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## 🚀 Getting Started

```bash
npm install
npm run dev
```
